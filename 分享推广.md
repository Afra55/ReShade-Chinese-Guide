# ReShade 中文指南上线：把 FRAMED 社区的硬核教程，完整搬到了中文世界

如果你玩过 PC 游戏截图、拍照模式或者后期调色，大概率听过 **ReShade** 这个名字——它几乎是游戏画面后期处理的标配工具。但真正好用的，往往不是「装上去就行」，而是：**怎么正确配置深度缓冲、怎么选着色器、怎么用 LUT、怎么在 UWP / Game Pass 游戏里注入**……

这些知识分散在英文社区里，查起来费时费力，对中文用户尤其不友好。

于是，我做了这件事：**把 [FRAMED. Screenshot Community](https://framedsc.com/) 的 ReShade 指南完整镜像、汉化，并部署为可在线阅读的中文版。**

---

## 这是什么？

**[ReShade-Chinese-Guide](https://github.com/Afra55/ReShade-Chinese-Guide)** 是一个面向中文用户的 ReShade 教程仓库，内容基于 FRAMED 社区整理，目前已完成 **13 篇指南** 的简体中文翻译，保留原有结构与配图，方便对照学习。

你可以把它理解为：**一份可直接查阅的 ReShade 中文手册**，而不是零散的论坛帖子。

---

## 里面有什么？

目前涵盖以下主题：

| 类别 | 内容 |
|------|------|
| 入门 | 安装与配置 ReShade、深度缓冲检查 |
| 着色器 | 完整着色器目录、电影感景深、高度雾 |
| 工作流 | LUT 生成、深度缓冲导出、RLE 长曝光 |
| 进阶 | UWP / Game Pass 游戏注入 |
| 插件 | ShaderToggler、MSADOF |

共计 **13 个页面、109 张配图**，侧栏导航、提示框、图片说明均已汉化。着色器名、游戏名等技术专有名词保留英文，方便你在 ReShade 界面里对照操作。

---

## 为什么做这个项目？

1. **ReShade 上手门槛不低**——深度缓冲、效果技法顺序、预处理器定义，任何一个环节配错都会导致效果异常。
2. **FRAMED 社区的指南质量很高**，但是英文，且分散在网站各个页面里。
3. **汉化后保留原结构**——图片路径、页面层级不变，后续更新、校对、补充都更方便。

内容版权归原作者及 FRAMED 社区所有，本仓库仅作整理与汉化，方便中文用户学习与查阅。
