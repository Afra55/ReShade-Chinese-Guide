安装与配置 ReShade

ReShade 是一款功能强大的后期处理系统，拥有丰富的功能。要充分发挥 ReShade 的潜力，关键在于针对所用游戏进行正确配置。本指南基于 ReShade v4.2 编写。如果您使用的是旧版本，部分内容可能仍然适用，但不做保证。

本指南面向截图用途。这意味着它不会深入介绍性能模式等话题，因为在设置截图画面及所需 ReShade 配置时，游戏通常处于暂停状态。

视频教程

如果您更喜欢视频教程，FRAMED 成员 TheGordinho Ochinchin 制作了一段视频，引导您完成基本步骤。

为游戏安装 ReShade

要在游戏未运行的情况下为其安装 ReShade，首先需要从 ReShade 官网下载安装程序。下载并运行 .exe 后，将打开一个小程序引导您完成安装流程。第一步只需选择游戏的 .exe 文件即可。

之后程序应会自动识别正确的渲染 API。如未能自动识别，对于较新的游戏请手动选择 DirectX 11。最后一步是下载一组着色器。如果您是第一次使用 ReShade 且尚未做任何配置，建议从 GitHub 上的主仓库下载所有着色器。如果您决定不下载任何着色器，至少请下载 DisplayDepth 着色器，因为它有助于正确配置 ReShade。

完成以上步骤后，ReShade 即已准备就绪。启动游戏时，屏幕顶部将显示一个横幅，提示 ReShade 已激活，并显示打开菜单所需的快捷键组合，下一节将用到它。

配置 ReShade

当 ReShade 在游戏中运行后，您可以打开其菜单。较新版本的 ReShade 使用 HOME 键打开菜单。如果您复制了别人的 ReShade 配置，可能需要使用 Shift-F2。

打开菜单后，如果您是第一次使用，可能会先显示教程。教程将为您设置一个预设，并介绍各部分的用途。如果您在安装 ReShade 时下载了着色器，将在主页选项卡上看到这些着色器包含的所有效果技法，按照一定顺序整齐列出。您可以右键单击技法名称，然后单击切换快捷键旁边的文本框，为每个效果技法设置快捷键。

按下切换快捷键即可启用或停用该效果技法。要启用/停用效果技法（例如 DisplayDepth），也可以用鼠标勾选技法名称前面的复选框。当某个效果技法处于激活状态时，其控件会显示在主页选项卡底部。您可以向上拖动水平蓝色分隔条来扩大显示区域。

将菜单解锁为浮动窗口

默认情况下，菜单停靠在屏幕左侧。如果您想查看被菜单遮挡的区域，这可能会造成不便。要使菜单可移动，您可以将某个选项卡（例如 DX11 选项卡）从菜单中拖出，然后将其他所有选项卡合并到该选项卡，从而创建一个浮动菜单。注意不要将选项卡停靠到屏幕边缘——当出现将选项卡停靠到某侧的图标时，请注意避免将选项卡拖放到该图标上。

您可以通过向左或向右拖动选项卡标题来重新排列浮动菜单中的选项卡，从而按照您希望的顺序排列各选项卡。单击菜单中的任意位置并拖动，即可移动菜单。

此后关闭并重新打开菜单，它将保持浮动状态，您可以将其拖动到任意位置。

执行顺序

主页选项卡上的效果技法看似随机排列，但实际并非如此。执行顺序为从上到下。这意味着，如果您同时启用了 CinematicDOF 和 LumaSharpen，且按从上到下的顺序排列，ReShade 将先执行 CinematicDOF，再执行 LumaSharpen。这样的效果是：CinematicDOF 的输出结果将作为 LumaSharpen 的输入。

最佳顺序取决于您的个人偏好，但一般规律是：任何偏向色彩处理的效果技法应在偏向镜头模拟的效果技法（例如景深着色器、胶片颗粒或叠加层着色器）之前执行。

设置全局预处理器定义

ReShade 使用若干全局预处理器定义，这些定义会影响效果技法的工作方式，例如深度缓冲的显示效果。您可以单击主页选项卡上效果技法参数顶部的蓝色"编辑全局预处理器定义"按钮来编辑这些定义。这些设置的含义在检查深度缓冲访问一节中有详细说明。

ReShade 设置

您可以在 ReShade 的设置选项卡中配置各种设置。对于截图爱好者而言，截图部分尤为重要。所用快捷键不应与您正在使用的其他覆盖层（如 Steam、UPlay 等）冲突，同时您应选择最适合自己的文件格式，例如 PNG 是无损格式，且可上传至任何平台。这部分内容相当浅显易懂。

调整效果技法的参数

一切准备就绪后，如何调整效果技法的设置？首先，通过按下您设定的切换快捷键，或勾选主页选项卡上技法名称前面的复选框来激活该着色器。您将看到 ReShade 在主页选项卡的蓝色分隔条下方显示该着色器的所有设置。

看到滑块时，可以用鼠标拖动进行调整。对于数值输入框，同样可以用鼠标操作：单击输入框后按住鼠标左键并向左或向右拖动即可改变数值。拖动滑块或输入框时按住 shift，数值的变化速度将加快。按住 ctrl 并单击，将弹出一个文本框，允许您直接输入数值。当您需要输入超出控件默认范围的数值时，此功能非常有用。

检查深度缓冲访问

为确保正确设置深度缓冲，关键是要安装 DisplayDepth 着色器。激活 DisplayDepth 着色器后，您将看到深度缓冲的完整预览，以及您所选择的预处理器定义值是否正确。ReShade 论坛上有一篇关于如何使用 DisplayDepth 着色器正确设置深度缓冲的精彩教程，建议阅读并正确配置您的 ReShade。每款游戏只需完成一次此设置。

选择正确的深度缓冲

游戏可能使用多个深度缓冲，或者在渲染最终画面时动态切换深度缓冲，从而导致深度缓冲看起来不可用。要告知 ReShade 使用哪个深度缓冲，您需要进入 ReShade 覆盖层中的一个额外选项卡。具体是哪个选项卡取决于您使用的 ReShade 版本。

ReShade v4.x 及更低版本：如果您使用的是 ReShade v4.x 或更低版本，ReShade 菜单/覆盖层包含一个额外选项卡，名称为 D3D9、D3D10、D3D11 或 D3D12 之一。
ReShade v5.x 及更高版本：如果您使用的是 ReShade v5.x，ReShade 包含一个名为 Add-ons 的额外选项卡。

此额外选项卡显示所有当前活动的深度缓冲。要确定应选择哪个，请先启用 DisplayDepth 着色器。您可能会看到白屏或黑屏。进入 ReShade 菜单/覆盖层中上述额外选项卡后，可以通过勾选复选框来选择正确的深度缓冲（或尝试勾选在清除前复制深度复选框）。如果屏幕显示恢复正常，则说明您选择了正确的深度缓冲，可以将其用于依赖深度缓冲的效果技法了。

着色器复制

有时您可能需要多次使用同一个着色器（CanvasFog 和 StageDepthPlus 着色器在这方面尤为常见）。对于大多数着色器，只需将着色器代码中的效果技法名称替换为另一个不同的名称即可，但某些着色器还需要更改 namespace 或纹理名称，请注意这一点。务必查看着色器代码中关于复制着色器的注释说明。

联网游戏中的深度缓冲

当游戏是联网游戏或持续进行网络访问时，深度缓冲将被禁用。由于许多效果都需要访问游戏的深度缓冲，ReShade 一旦检测到游戏为联网游戏（即存在持续的网络流量），便会主动禁用深度缓冲访问权限。如果因此导致深度缓冲不可用，依赖深度缓冲的着色器很可能将无法正常工作。

因此，如果您尝试在持续使用网络连接的游戏中使用 ReShade（无论是出于 DRM 保护，还是因为游戏本身需要始终在线，例如《全境封锁》或《Steep》），您将需要使用自定义 ReShade .dll 版本才能获得深度缓冲访问权限，否则 ReShade 将阻止访问。当然，您也可以在没有深度缓冲访问权限的情况下使用 ReShade，但依赖深度缓冲的着色器（如 CinematicDof）将无法使用。

请注意，部分游戏的反作弊系统已将官方版本的 ReShade 列入白名单，因此使用自定义 .dll 可能会导致您被封号，具体情况因游戏而异，请事先做好调查研究。

与其他可注入模组同时使用 ReShade

有时您可能会遇到无法将 ReShade 与另一个同样注入进程的模组同时使用的情况。解决此问题的一种方法是使用 ASI Loader，将额外的模组放入 scripts 文件夹。请阅读 ASI Loader 仓库中的 README 文件，了解更多使用方法。
